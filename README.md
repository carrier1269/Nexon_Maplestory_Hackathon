## Nexon-Likelion-Maplestory-Hackathon

### [Nexon Maplestory worlds 개발자 지원 홈페이지](https://techit.education/courses/technology/superhackathon)

### [메이플스토리 API 개발전용 사이트](https://maplestoryworlds-developers.nexon.com/ko)

#### 코딩분야에 호기심이 많다보니까, 내가 하는 게임에 대해서는 어떻게 개발이 이루어질까? 라는 의문점을 가지고 게임 개발에 도전해보았습니다.
#### 평소 Python을 다루다 보니까 maplestory 개발 언어인 lua script 언어를 사용할 줄 모르면 어떡하지? 그리고 아무래도 게임 개발에 관심과 능력이 있는 인원들에 대한 이력서를 1차 서류 심사를 통해 선발하다 보니까 많이 걱정을 했었는데 생각보다 lua script 언어와 Python 언어의 공통점이 많았고 (self 선언을 통한 class간의 data 주고받기, c++ & java와 다르게 python처럼 ;세미콜론을 사용하지 않는다는 점, 다양한 라이브러리에 대한 API 자료가 많고 API 사용법은 되게 직관적이라는 점)
#### API를 참조하면서 스킬, 타격effect, 사운드, hp 계수, 마나 관리 계수, user동작에 대한 커스터마이징, 게임 server에 대한 server & client 분리 및 관리를
#### 하나하나 진행할 때 마다 오기가 생기기 시작했고, 4달동안 재미있게 진행했던 프로젝트입니다.

#### 저는 이 프로젝트에서 user attack 부분의 스킬, 타격 effect, 사운드 생성을 담당했고,
#### 다른 팀원분들이 만들어 주신 boss map에 대해서 
#### 퀘스트 진행도에 따른 to do list 설정 및 map 간의 portal 사용가능 & 불가능 및 이동경로를 설정하고, 
#### 퀘스트 진행도에 따른 월드내의 npc와의 대화 설정,
#### 월드내의 entity에 대한 components들에 대한 유지 보수 및 전체적인 흐름에 대한 제어등을 담당하였습니다.

#### 결과로는 수상이 아니였지만, 처음으로 접해본 언어에 대해 잘 적응하고 다양한 문제들을 해결해봄으로써 능력을 한 단계 상승시킬수 있는 기회라고 생각하였습니다.
